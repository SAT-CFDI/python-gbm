Installation
____________________

Install GBM API from PyPI with:

.. code-block:: sh

    python -m pip install gbm

or install from source with:

.. code-block:: sh

    git clone https://github.com/SAT-CFDI/python-gbmapi
    cd python-gbmapi
    python -m pip install .
